# Neon Retro Flutter Shell

This folder contains the Android shell for Neon Retro.

## Role

The React layer controls the browser experience. The Flutter shell owns the phone-side browser engine and native features.

## Current bridge commands

- LOAD_URL
- GO_BACK
- GO_FORWARD
- RELOAD
- CLEAR_CACHE
- SET_DESKTOP_SITE

## Next native features

- downloads
- cookies
- cache cleanup
- site permissions
- notifications
- location simulation
- local React bundle loading
